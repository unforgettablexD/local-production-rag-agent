# Product FAQ

## Supported Integrations
The platform currently supports Slack, Microsoft Teams, Google Drive, Jira, and GitHub integrations. Salesforce integration is planned but not yet generally available.

## Pricing Basics
The Professional plan includes unlimited chat history, role-based access controls, and email support. The Enterprise plan adds SSO, audit logs, and a dedicated customer success manager.

## Model Hosting
Customers can deploy with either a managed cloud LLM or a self-hosted local model stack depending on their security requirements.
